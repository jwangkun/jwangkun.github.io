Centos 卸载重装ssh

Posted on 2021-09-22，4 min read

可以通过这条命令查看卸载前后的openssh相关服务安装情况

rpm  -qa  openssh*

卸载

yum remove openssh*

然后安装ssh


yum install openssh-server
